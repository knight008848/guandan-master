# 战大师 (Guandan Master) 🃏

掼蛋大师是一款运行在浏览器端的精品掼蛋（Guandan）棋牌游戏。项目采用现代感十足的**毛玻璃（Glassmorphism）视觉设计**、高分辨率纯 CSS/SVG 扑克卡牌样式以及细腻的 CSS/JS 动画音效联动，为玩家在电脑端提供真实的沉浸式对局体验。

本项目不需要任何后端依赖，完全由原生 **HTML5 + Vanilla CSS + ES6 Javascript** 构建，可以直接在浏览器中双击 `index.html` 运行！

---

## 🌟 项目亮点与特色

### 1. 极致精美的视觉设计
- **拟物化绿色呢面牌桌**：采用高拟真径向渐变，附带柔和的内阴影，使出牌区和牌局视感非常饱满。
- **高清现代感卡牌**：利用纯 CSS 打造高清卡牌边框、自适应字号以及圆润阴影。
- **逢人配流光特效**：作为红心主牌的“逢人配”拥有独特的金色外发光和“配”字角标，在理牌时醒目呈现。
- **玻璃态半透明 UI**：游戏顶栏、操作控制区、结算面板和进贡对话框均采用毛玻璃效果。

### 2. 动感流畅的物理动效
- **发牌动效**：卡牌以抛物线轨迹、旋转及缩放效果从牌桌中央依次飞入每个玩家的牌区，伴有流畅的物理惯性。
- **炸弹爆炸粒子特效**：打出炸弹（或同花顺、天王炸）时，触发整个牌桌**屏幕轻微震动**，并从出牌中心喷射出数十个色彩斑斓的粒子光斑，呈扇形消逝。
- **出牌滑行动效**：出牌或过牌（PASS）均伴有滑行进入出牌堆的缓动动画。

### 3. 完整的掼蛋规则链条
- **进贡与退贡**：严格遵循掼蛋规则。局前自动判定是否抗贡（双王抗贡），并支持玩家自主点击卡牌进贡、接收退贡；AI 队友及对手具备完备的自动进贡和退贡挑选机制。
- **逢人配（万能牌）智能拆解**：自动支持红心主牌作为逢人配，配合玩家选中的任意牌型，穷举得出能够组合的最高优先级牌型（如填补顺子中的空缺、配对、配炸弹等）。
- **接风规则**：当某玩家出完最后一张牌成为本轮最大且所有对手 PASS 时，其队友自动获得下一轮的“接风首发权”。
- **晋级与段位攀爬**：支持从打 2 升级到打 A 的积分机制。根据获胜顺序（双游 +3 级，头游+三游 +2 级，头游+末游 +1 级）进行晋级升级。

### 4. 舒适的玩家交互
- **鼠标滑动多选**：支持在手牌区按住鼠标左键横向划过，一次性多选/取消选择多张卡牌，操作极其丝滑。
- **智能提示**：根据上家的牌型和点数，智能在玩家手牌中定位能够压制的最优组合，循环展示。
- **快速整理与重置**：一键重置选择，或以“大王 > 小王 > 主牌 > A > ... > 2”的方式一键排选手牌。

## 🎮 玩法指引与运行

### 如何开始？
1. **直接运行**：使用任何现代浏览器（Chrome, Edge, Firefox, Safari 等）直接双击项目目录下的 [index.html](file:///C:/Users/xx/.gemini/antigravity-ide/scratch/guandan-master/index.html) 即可启动游戏。
2. **重置/下一局**：牌局结束后会自动结算并弹窗提示得分，点击“开始下一局”可自动过渡到下一局（级数已升，并进入进贡/退贡界面）。

### 交互操作：
- **选牌**：点击任意卡牌即可使其向上浮起（选中）；再次点击可降下。
- **滑动多选**：按住鼠标左键，横向拖动滑过一排卡牌，可以直接批量选中。
- **操作面板**：
  - `出牌`：打出当前选中的牌（系统会自动检测是否符合大过上家的条件）。
  - `不出`：跳过当前回合（上家不是自己时可用）。
  - `提示`：自动在您手牌中寻找可以压制的组合。
  - `重置选择`：取消所有卡牌的悬浮选定。
  - `整理手牌`：按照规则大小顺序重新排布手牌。
